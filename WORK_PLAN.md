# 旅遊 / 出差行程規劃器（7 天）— 工作計畫 & SOP 勾選清單

目標：完成一個前端單頁網站，能新增 / 編輯 / 刪除行程，支援「日期 + 時間」選擇並自動排序；可輸入行程名稱、地點、交通方式、備註；備註欄支援輸入網址並可選取套用為超連結；預設以 7 天行程視圖呈現；介面風格調整為「少女可愛風」，並支援手機優先查看看板。

交付物（檔案切分）：
- `index.html`
- `style.css`
- `app.js`
